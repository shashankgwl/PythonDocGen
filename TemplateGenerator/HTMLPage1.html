Country/Region

All

 

Persona (Actor)

System Process

 

Description

Key Requirement is to be able to look into Outlook Schedule for a particular  resource and determine if the resource is available or not.  Do not need to know why the resource is not available.


Direction: From PSA to Outlook

The resource bookings assigned to an engineer should be uploaded to the assigned engineer’s Outlook calendar.


Trigger Events (What event makes this happen)

  1. When a new booking is created and assigned to a resource.  
  2. When a current resource is modified and assigned to a different resource.

Solution: 

OOB integration from resource bookings to Outlook calendar.

 

Preconditions (What is needed before this can happen)

1. The field “Synchronize resource bookings with Outlook” must be checked under the System Settings.

 

2. The resource must have the field “Synchronize resource bookings with Outlook” checked in the personal options.

 

Post Conditions (What happens after the event happens/what is everything affected by change)

The booking will display in the resource’s Outlook calendar.
